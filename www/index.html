Log4j Detector

FOX InfoSec is proud to partner with Praetorian to develop this utility to help identify instances of vulnerable Log4j classes in running Java processes.

 

Downloads

  • Windows (32-bit)
  • Windows (64-bit)
  • Linux (32-bit)
  • Linux (64-bit)

Usage

  • Run the binary as SYSTEM / Administrator (Windows) or as root (Linux). Data will be sent to the reporting server automatically.
